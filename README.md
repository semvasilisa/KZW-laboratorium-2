# Opis plików

- **witi.cpp**  
  Zawiera pierwszą wersję implementacji algorytmu. W tej wersji zastosowano podejście rekurencyjne wykonywane w pętli, bez dodatkowych optymalizacji.

- **witi2.cpp**  
  Zawiera końcową, zoptymalizowaną wersję algorytmu. W implementacji wykorzystano technikę memoizacji, co pozwala znacząco przyspieszyć obliczenia poprzez zapamiętywanie wcześniej wyznaczonych wyników.
